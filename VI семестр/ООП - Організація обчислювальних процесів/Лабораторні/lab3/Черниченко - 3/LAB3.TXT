                        ЛАБОРАТОРНАЯ РАБОТА #3


Задание на работу
~~~~~~~~~~~~~~~~~

1. Согласно варианту выполнить задание.


Варианты (выбираются по номеру зачетной книжки)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

 1. Определить вероятность появления обязательных назначений при
    изменении заполненности матрицы "1" связности от 1% до 100% для
    заданной размерности от 10 до 30.

 2. Определить вероятность появления конфликтных назначений при
    изменении заполненности матрицы "1" связности от 1% до 100% для
    заданной размерности от 10 до 30.

 3. Составить программу реализации венгерского метода для неоднородной
    вычислительной системы.

 4. Составить программу реализации метода "ветвей и границ" для
    неоднородной вычислительной системы.

 5. Составить модель пространственного планировщика на принципе
    исключающего планирования.

 6. Составить модель пространственного планировщика на принципе
    не исключающего планирования.

(7). Реализовать модель пространственного планировщика на основе
(7)  алгоритма преобразования матрицы связности с выстраиванием "1"
(7)  по главной диагонали.

 8. Составить программу реализации венгерского метода для неоднородной
    вычислительной системы с ограничениями реального времени.

 9. Составить программу построения базового решения для взвешенного
    ациклического графа (без пересылок).

10. Составить программу построения базового решения для взвешенного
    ациклического графа (с пересылками).

11. Составить программу выделения всех критических вершин для заданного
    взвешенного ациклического графа по алгоритму Симоненко.

12. Определить временную сложность алгоритма выделения всех критических
    вершин для заданного взвешенного ациклического графа по алгоритму
    Симоненко.

13. Составить программу приведения графа, заданного в виде матрицы
    связности к ярусно-параллельной форме.

14. Составить программу выделения вершин имеющих признак явной транзитности.

15. Составить программу выделения вершин имеющих признак не явной
    транзитности.

16. Составить программу определения (локализации) зоны поиска решения для
    графа без пересылок. 

17. Составить программу определения (локализации) зоны поиска решения для
    графа с пересылками. 

18. Составить программу распределения задач по процессорам с веерным
    распределением от самого широкого яруса.
