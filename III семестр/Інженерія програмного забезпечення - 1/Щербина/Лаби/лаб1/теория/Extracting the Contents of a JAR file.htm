Выделение содержимого из JAR-файла

Базовый формат команды для выделения содержимого JAR-файла такой: 
jar xf jar-файл [архивный-файл(ы)]

Давайте рассмотрим опции и аргументы этой команды:

  • Опция x показывает, что вы хотите выделить (extract) файлы из архива JAR.
  • Опция f показывает, что JAR-файл, файлы из которого должны быть выделены, задан в командной строке, а не в стандартном потоке ввода.
  • Аргумент jar-file - это имя файла (или маршрут и имя) для JAR-файла, из которого выделяются файлы.
  • архивный-файл(ы)(s) - необязательный аргумент, состоящий из списка с разделителем-пробелом файлов, которые должны быть выделены из архива. Если этот аргумент отсутствует, утилита Jar будет выделять все файлы из архива.

Как обычно, порядок, в котором в команде употребляются опции x и f, не имеет значения, но между ними не должно быть пробелов.

При выделении файлов утилита Jar делает копии требуемых файлов и записывает их в текущий каталог, воспроизводя ту структуру каталогов, которая была в архиве. Исходный JAR-файл не изменяется.

Внимание: При выделении файлов утилита Jar перезаписывает все существующие файлы, имеющие то же полное имя, что и выделяемые файлы.

Пример

Давайте выделим некоторые файлы из JAR-файла TicTacToe, который мы использовали в предыдущих разделах. Вспомните, что содержание TicTacToe.jar такое: 
META-INF/MANIFEST.MF
TicTacToe.class
audio/
audio/beep.au
audio/ding.au
audio/return.au
audio/yahoo1.au
audio/yahoo2.au
images/
images/cross.gif
images/not.gif

Предположим, вы хотите выделить файл класса TicTacToe и файл изображения cross.gif. Чтобы сделать это, вы можете использовать команду: 

jar xf TicTacToe.jar TicTacToe.class images/cross.gif

Эта команда делает две вещи:

  • Помещает копию TicTacToe.class в текущий каталог.
  • Создает в текущем каталоге каталог images, если он еще не существует, и копирует в него cross.gif.

Исходный JAR-файл TicTacToe остается неизменным.

Таким путем из JAR-файл может быть извлечено сколько угодно файлов. Если в команде не указано, какие файлы выделять, утилита Jar выделяет все файлы из архива. Например, вы можете выделить все файлы из архива TicTacToe, используя такую команду: 

jar xf TicTacToe.jar
