# Т-образная эквивалентная схема для схемы транзистора с ОЭ
  [рис. 7.1]
  Для включения транзистора с ОЭ, Т-образная эквив. схема должна иметь одинаковые параметры в режимах холостого хода и короткого замыкания с малосигнальной эквивалентной схемой

  alpha*I_э*R_к = beta*I_б*R_к
  I_э ~= I_б => alpha*R_к ~= beta*R_к => R*_к ~= beta*R_к/alpha
  beta = 1/(1 - alpha) => 

  В схеме с ОЭ, активное и ёмкостное сопротивление коллекторной цепи в (beta+1) раз меньше, чем для транзистора с ОБ.

  Из-за трудности учёта и контроля реальных физических явлений транзистора, расчёт по эквивал. схемам не всегда обеспечивает требуемую точность, поэтому очень часто транзистор представляют в виде четырёхполючника и заменяют физические параметры практическими эквивалентными схема в h-параметрах и y-параметрах. 

## h-параметры
  delta U_1 = h_11*delta I_1 + h_12*delta U_2
  delta I_2 = h_21*delta I_1 + h_22*delta U_2

  Коэфициенты уравнений определяются экспериментально с помощью короткого замыкания на выходе и обратного холостого хода на входе транзистора.

### h-параметры для транзистора с ОЭ
  Короткое замыкание на выходе транзистора [U_кэ = 0] :
    h_11э = (delta U_бэ / delta I_б) = R_вх_э = B_вх_э*(beta+1) — входное сопротивление
    h_21э = (delta I_к / delta I_б) = beta — коэфициент передачи тока

  При обратном холостом ходе на входе транзистора [I_б = 0] :
    h_21э ~= 0 — величина, обратная коэфициенту усиления по напряжению
    h_22э = I/R*_к — величина, обратная выходному сопротивлению (выходная проводимость)

#### h-параметры для расчёта низкочастотных схем с ОЭ
  Т.к. h_21э ~= 0, то эта величина в расчётах не учитывается. В большинстве случаев, небольшие приросты синус. напряжений и тока заменяются комплексными величинами:
    U'_бэ = h_11*I'_б + h_12*U'_кэ
    I'_к = h_21*I'_б + h_22*U'_кэ

  [рис. 7.2]

  h_21*I'_б — зависимый источник тока, характеризующий зависимость коллекторного тока от тока базы. Фактически, он показывает усиление базового тока.  
  Аналогично, h_12*U'_кэ — зависимый источник ЭДС, характеризующий обратную связь по напряжению.

## y-параметры
  Используется для расчётов высокочастотных схем. В этом случае, независимыми параметрами являются напряжения.
    delta I_1 = y_11*delta U_1 + y_12*delta U_2
    delta I_2 = y_21*delta U_1 + y_22*delta U_2

### y-параметры для транзистора с ОЭ
    delta I_б = y_11э*delta U_бэ + y_12э*delta U_кэ
    delta I_к = y_21э*delta U_бэ + y_22э*delta U_кэ

  Коэфициенты усиления при прямых и обратных токах определяются при коротких замыканиях [U_кэ = 0]:
    y_11э = 0         y_12э = 0
    y_21э = 0         y_22э = 0

#### h-параметры для расчёта низкочастотных схем с ОЭ
  После замены небольших приростов комплексными величинами получим
    [рис. 7.3]

  y_12*U'_кэ — зависимый источник от напряжения в другой ветке
  Аналогично, y_21*U'_бэ — зависимый источник от входного напряжения

  Структура биполярного транзистора определяется его динамические свойства. Конечное время движения носителей заряда через базу создаёт эффект запаздывания сигнала, что равносильно выходной (?)мощности.

  Пусть на базе транзистора задан отрицательный отричательный потенциал, ключ в "1", тогда эммитерный переход закрыт за счёт суммы напряжения U_1 и контактной разности потенциалов
