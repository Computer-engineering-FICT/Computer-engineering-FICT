# Схемотехника усилительных устройств
  Усилительный каскад — однотипные ячейки, из которых строят усилители. У.к. может содержать несколько транзисторов, однако его нельзя разделить на более простые компоненты без утраты его специфических свойств. 

## Усилительный каскад по схеме с ОЭ
  [рис. 11.1]

  U_вых = i_к * R_к
  U_вых = i_к * R'_к, R'_к = (R_к * R_н) / (R_к + R_н)
  i_к = i_б * h_21э = h_21э * U_вх / R_вх

  При параллельном подключении, фазы входного и выходного сигнала отличаются на pi — инвертирующий каскад.

  R_б — балластное сопротивление, служит для линеализации характеристики, R_б >> R_вх
  i_б = i_б * h_21э = (U_вх * h_21э) / (R_б + R_вх) ~= h_21э * U_вх / R_б

  Введение R_б уменьшает коэф. усиления каскада.

  Входное сопротивление на начальном этапе ВАХ имеет существенно нелинейный характер, поэтому одинаковые приросты базового тока обусловлены разными приростами входного сигнала.

  Передаточная характеристика каскада преобретает нелинейность, что усложняет практическое использование каскада

  В выходной характеристике каскада действуют U_сиг и U_см, задающее требуемый режим работы.

# Схема и режим покоя
  Р.п. — Режим отсутствия сигнала
  [рис. 11.2]

  Для усилителей постоянного тока характерно двухполярное питание с напряжениями +E_к и -E_э. Пусть входной сигнал U_вх = 0, тогда в схеме протекают постоянные состовляющие токов, обусловленные E_к и E_э. При наличии входного сигнала, к постоянным составляющим добавляются переменные компоненты, пропорциональные величине U_вх. В рабочем режиме, полные значения напряжения и тока:

  U = U0 + delta U
  I = I0 + delta I

  [рис. 11.3]

  U_вх = 0,
  I0_э = (E_э - U*) / (R_э + (1-alpha)*R_б)

  I0 задаются заранее и их совокупность определяет режим работы (рабочую точку транзистора) в режиме покоя. В качестве входных данных можно задать E_э и R_э, это делается с точки зрения стабильности рабочей точки транзистора.

  R_э >> (1-alpha)*R_б
