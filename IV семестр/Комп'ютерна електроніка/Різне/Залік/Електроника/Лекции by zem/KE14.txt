Синфазный сигнал. Идеальный Д.у. реагирует только на диф. сигнал
Посколько диф. сигнал делится поровну между двумя плечами, потенциал средней точки (потенциал эммитера), для упрощения анализа, остаётся неизменным и считается нулевым.

Любую комбинацию напряжений delta U_б1 и delta U_б2 можно представить в виде суммы синфазной и диф. составляющей сигнала.
    delta U_б1 = U_вх_с + 1/2 U_вх_д
    delta U_б2 = U_вх_с - 1/2 U_вх_д

Действие каждой из составляющих можно рассматривать раздельно.

Синф. и диф. составляющие сигнала:
    U_вх_с = 1/2 (delta U_б1 + delta U_б2)
    U_вх_д = delta U_б1 - delta U_б2

Выходное напряжение можно представить как сумму синф. и диф. составляющих:
    U_вых_с = 1/2 (delta U_k1 + delta U_k2)
    U_вых_д = delta U_k1 - delta U_k2

Существенную роль в работу Д.у. играет постояннство тока I0.

Если источник идеальный (R_i => inf), то синф. составляющая вызывает только прирост эммитерного сигнала и токи в обоих плечах остаются неизменными.
    delta U_э = delta U_б = delta U_вх_с

Если R_i ≠ inf, то
    delta I = delta U_э / R_i

Если параметры идентичны, то delta U_k1 = delta U_k2

## Параметры дифференциального усилителя
    \system
    U_вых_с = K_сс * U_вх_с + K_сд * U_вх_д
    U_вых_д = K_дс * U_вх_с + K_дд * U_вх_д
    \endsystem

    В идеальном Д.у., K_дс = K_сд =0

### Коэф. диф. составляющей (K_дд)
    Усиление каждого плеча можно рассматривать как усиление отдельного усилителя:
        K = - (alpha * R_к)/(R_э + r_э + (1-alpha)(R_б + r_б))

    В каждом плече усиливается только половина входного сигнала, а на выходе усиленные сигналы суммируются, коэф. усиления Д.у. равен коэф. усиления отдельных плеч.

    Если R_э = 0, то:
        K = - (alpha * R_к)/(r_э + (1-alpha)(R_б + r_б)) [именно минус]

    В случае низкоомных источников сигнала, R_б < 1 кОм, тогда:
        K ~= -(alpha * R_к)/r_э

    Усиление от темпаратуры
        K = (E_k - U0_k) / phi_t

    Таким образом, при заданном напряжении (?) тока коллектора, коэф. усиления не зависит от рабочего тока.

### Коэф. синф. составляющей (K_сс)
    K_сс = U_вых_с / U_вх_с при U_вх_д = 0
    delta U_э = U_вх_с

    Если R_i, то delta I0 = U_вх_с / R_i

    K_сс = - (alpha * R_к)/(2*R_i)

    R_k / R_i << 1 => K_сс < 1

### Коэф. синф.-диф. составляющей (K_сд)
    K_сд = U_вых_с / U_вх_д при U_вх_с = 0

    K_сд меняется, в основном, из-за неидентичности плеч усилителя (наличие R_i). Тогда:
        K_сд = delta K = K_1 - K_2

    Для оценки K_сд:
        K = (K_1 + K_2) / 2
        delta K / K = delta R_k / R_k
        K_сд = K*(delta R_k / R_k)

    Изменение постоянной составляющей коллекторных потенциалов под действием диф. сигнала в десятки раз меньше выходного напряжения.

### Коэф. подавления синф. составляющей (K_п)
    Анализ начальной системы показывает, что влияние синф. составляющей входного сигнала на диф. составляющую значительно из-за того, что K_дс * U_вх_с и K_дд * U_вх_д неотличимы от входного сигнала.

    K_п = 20*lg |K_дс / K_дд| — коэф. подавления синф. составляющей
    K_п = K_дд / K_сд

    Как правило, самым распространнёным источником диф.-синф. сигнала являются помехи.

    Пусть R_i, неидентичность параметров определяется соотношением
        alpha_1 = alpha + delta alpha
        alpha_2 = alpha - delta alpha

        R_K1 = R_K + delta R_K
        R_K2 = R_K - delta R_K

        delta I0 = U_вх_с / R_i. Пусть этот прирост равномерно делится между двумя плечами:
            delta U_k1 = -alpha_1*(U_вх_с / 2*R_i)*R_k1
            delta U_k2 = -alpha_2*(U_вх_с / 2*R_i)*R_k2

        K_дс = - alpha (R_к / R_i) ((delta alpha / alpha) + (delta R_k / R_k))

    K_пп = (1 / delta) * (R_i / R_э)
    delta = (delta alpha / alpha) + (delta R_k / R_k) — коэф. асимметрии
