# Типовые функциональные каскады полупроводникового усилителя
  Соединение усилителей, обеспечивающее последовательное усиление сигнала до требуемой величины
  Функционально различают усилители:
    * Предварительный — обеспечивает непосредственную связь источника сигнала и усил. устройства. Главное требование — минимальное ослабление выходного сигнала (большое R_вх)
    * Промежуточный — буфер между предварительным и выходным усилителями, согласующий их сигналы.
    * Выходной — предназначен для получения мощности, обеспечивающий работоспособности вых. устройства.
    Главная характеристика — КПД.

# Нелинейные электрические цепи
  Нелинейный элемент — его основной параметр зависит от тока, протекающего через данный элемент, либо напряжения на его выходах.

  [рис. 10.1]

  R_дин = dU/dI = delta M / delta I
  R_стат = U(п)/I(п) = m_U/m_I * tg(alpha)

## Методы расчёта
### Метод де?
  Замещение нелинейного элемента эквивалентной линейной схемой, справедливой для огрниченного диапозона изменения тока и напряжения в нелинейном элементе. Одна из реализаций — метод кусочно-линейной аппроксимации.

### Метод кусочно-линейной аппроксимации

  [рис. 10.2]

  CD — прямой участок, U_пр = U_до + I * R_д
  EA — обратный участок
  AB — участок пробоя

### Метод эквивалентных преобразований
  Группа нелинейных элементов заменяется одним эквивалентным (принцип суперпозиции).

  [П. 10]

### 
  Метод используется когда нет возможности проанализировать цепь методом эквивалентных преобразований и позволяет исследовать процессы в цепи как при изменении параматров R_1, R_2, так и при изменении внешнего напряжения

  [рис. 10.3]

  В основе метода — предположение о том, что суммарное напряжение на последовательно включённых элементах определяется внешним источником и не зависит от тока, протекающего в цепи.

  I = I_R1 = I_R2
  U_1(I) + U_2(I) = U_п
