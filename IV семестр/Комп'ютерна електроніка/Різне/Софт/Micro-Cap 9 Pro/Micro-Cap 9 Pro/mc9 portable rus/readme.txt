
Портабельные версии Micro-Cap 9.0.7.0 сделаны с помощью Xenocode Virtual Application Studio 2010. 

Несмотря на малый объём итогового файла приложения (около 19 МБт) портабельные версии содержат полный вариант Micro-Cap 9.0.7.0 - все библиотеки, все примеры, файлы помощи и руководства пользователя.

При запуске программы в текущей папке создаётся "песочница" (папка с именем Sandbox), в которой будут храниться все изменения файлов и настроек, которые происходят при работе портабельной версии. Если Вам не нужно хранить эти изменения - данную "песочницу" можно спокойно удалять после закрытия Micro-Cap.

Исполняемый файл портабельной версии поддерживает следующие параметры командной строки:
  mc9 - непосредственно запуск Micro-Cap 9.0.7.0 (используется по умолчанию)
  help - запуск справки по Micro-Cap   (mc9.hlp)
  reference - открытие полного справочного руководства Reference Manual   (RM.pdf)
  guide - открытие краткого руковододства пользователя User Guide   (UG.pdf)
