О средстве просмотра PowerPoint

С помощью средства просмотра Microsoft Office PowerPoint можно просматривать презентации, созданные в программах Microsoft Office PowerPoint 2000, PowerPoint XP, PowerPoint 2003 и PowerPoint 2007. Кроме того, средство просмотра PowerPoint позволяет открывать презентации PowerPoint, защищенные паролем.

ПРИМЕЧАНИЕ. Для просмотра презентаций, созданных в PowerPoint 2007, необходимо установить пакет совместимости Microsoft Office для форматов файлов Word, Excel и PowerPoint пакета Office 2007. Дополнительную информацию можно получить на веб-узле http://office.microsoft.com/ .

Требования к системе

  • Персональный компьютер с процессором с тактовой частотой 200 МГц или выше
  • Microsoft Internet Explorer 5 или более поздней версии
  • Поддерживаемые операционные системы: Microsoft Windows 2000 с пакетом обновления Service Pack 4, Windows XP с пакетом обновления Service Pack 1, Windows Server 2003 или Windows Vista
  • Оперативная память:
    • Для Windows 2000 с пакетом обновления Service Pack 4: 64 Мб оперативной памяти
    • Для Windows XP с пакетом обновления Service Pack 1, Windows Server 2003: 128 Мб оперативной памяти
    • Для Windows Vista: 512 Мб оперативной памяти
  • 5 Мб свободного места на жестком диске
  • Монитор Super VGA с разрешением 800 x 600 или выше

Проблемы поддержки языков

Средство просмотра PowerPoint не поддерживает средств проверки правописания Microsoft Office.

Средство просмотра PowerPoint не поддерживает презентации, созданные в PowerPoint 97 на двух языках.

Назад к Содержанию

Работа со средством просмотра PowerPoint

В средстве просмотра PowerPoint неправильно отображается анимация диаграмм. Вместо анимации на экране появляется последний кадр диаграммы

Средство просмотра PowerPoint не позволяет открывать презентации, защищенные с помощью функции IRM (Information Rights Management).

Средство просмотра PowerPoint не поддерживает запуск макросов и программ и не открывает связанных или внедренных объектов.

Средство просмотра PowerPoint не поддерживает отображение связанных с презентацией графических файлов следующих форматов: .eps, .pct, .emz, .cdr, .cgm, .wpg и .wmz.
